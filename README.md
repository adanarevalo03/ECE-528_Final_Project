# INTRODUCTION
This is our final project for ECE 528, where we demonstrated a line-following rover with a Bluetooth-controlled arm using the MSP432 microcontroller. The rover uses an 8-channel QTRX reflectance sensor array to detect and follow a line while also allowing the user to manually control the arm through Bluetooth communication.
The arm is controlled using the Adafruit Bluefruit LE UART module and Adafruit’s GUI application. The GUI allows the user to send commands to the rover and control the movement of the arm. The arm uses three servo motors controlled by PWM (Pulse Width Modulation) signals generated by Timer_A2 on the MSP432. CCR0 was configured to generate the 20 ms PWM period required for standard servos, while CCR1, CCR2, and CCR3 were used to independently control the arm, wrist, and claw movements.
The GUI also includes four buttons for additional rover controls. The first two buttons are used to open and close the claw, while the other two buttons are used to stop the rover or make the rover turn around and continue moving forward. Directional controls in the GUI allow the user to move the arm up, down, and adjust the wrist position of the claw.
The line-following algorithm used for the rover is a basic finite state machine (FSM) that relies on simple if statements and sensor readings from the reflectance sensors. The rover continuously checks the sensor values and adjusts its movement left or right whenever it moves away from the line. Different states were implemented to handle straight movement, slight turns, hard turns, dead ends, and T-intersections.

# Background & Methodology
This project uses several embedded systems concepts learned during the semester, including GPIO, timers, interrupts, PWM, UART communication, and sensor interfacing. The final result is a line-following rover with a Bluetooth-controlled arm using the MSP432 microcontroller.

The rover uses an 8-channel QTRX reflectance sensor array to detect and follow a line. A simple FSM (Finite State Machine) algorithm was implemented to help the rover move left or right depending on the sensor readings. The arm is controlled wirelessly using the Adafruit Bluefruit LE UART module and Adafruit’s GUI application.

The arm uses three servo motors controlled with PWM signals generated by Timer_A2. CCR0 was used to create the 20 ms PWM period, while CCR1, CCR2, and CCR3 controlled the arm, wrist, and claw movements. Timer_A0 was used for the DC motor PWM signals, and Timer_A1 interrupts were used to periodically read the reflectance sensors.

## The Peripherals Used:
1. GPIO for sensor inputs and outputs
2. UART for Bluetooth communication
3. Timer_A0 for DC motor PWM control
4. Timer_A1 for interrupts and sensor updates
5. Timer_A2 for servo motor PWM control
6. Reflectance sensors for line following
7. Adafruit Bluefruit LE UART module for wireless control

# Analysis & Results

The project was successful, and the main components were implemented to create a fully functional line-following rover with a Bluetooth-controlled arm. The rover was able to follow a line using the reflectance sensors while also allowing the user to control the arm wirelessly through the Adafruit Bluefruit GUI application.

The arm successfully controlled the arm movement, wrist movement, and claw operation using three servo motors driven by PWM signals from Timer_A2. The rover was also able to respond to different sensor readings by moving left, right, forward, or turning around based on the FSM algorithm.

One challenge during development was tuning the reflectance sensor values to improve line-following accuracy because it would not read the T-intersection. Another issue was adjusting the PWM duty cycles for the servo motors to achieve smoother arm movement. These challenges helped improve understanding of timers, interrupts, PWM generation, and debugging.
